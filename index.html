Ermöglicht automatische Erkennung von Beleg-Bereich, Name, Kategorie und Datum per Foto.

🛠 Einmaliges Supabase Setup

Im Supabase SQL Editor ausführen:

CREATE TABLE belege ( id BIGINT GENERATED ALWAYS AS IDENTITY PRIMARY KEY, name TEXT, datum DATE, ordner TEXT, kategorie TEXT, storage_path TEXT, created_at TIMESTAMPTZ DEFAULT NOW() ); ALTER TABLE belege DISABLE ROW LEVEL SECURITY;

Dann unter Storage → New Bucket einen Bucket namens belege-fotos anlegen (Public ✅) und dessen RLS deaktivieren.
